## Quick Actions

### Start Writing Immediately
1. Open app
2. Type in the Quick Editor at the top
3. Click "Create Note" when done

### Navigate Like a Pro
Press `Cmd/Ctrl+P` (or click search icon in header) to open the **Command Palette**:
- Type to search for any file
- Access any feature instantly
- Navigate with keyboard arrows

### Keyboard Shortcuts
- `Cmd/Ctrl+P` - Command Palette (quick switcher)
- `Cmd/Ctrl+E` - New Note
- `Cmd/Ctrl+K` - Focus Search
- `Cmd/Ctrl+G` - Knowledge Graph
- `Cmd/Ctrl+J` - Daily Journal

### Access Your Files
Click the **arrow button** on the left edge of the screen to:
- Browse all files
- Upload files
- Filter by tags
- Drag and drop to organize

### Edit In-Place
When viewing a file:
- Click **"Edit"** button (top right)
- Editor opens right there - no page reload!
- See Table of Contents while editing
- Click "Cancel" to go back to reading

## Getting Help

### Command Palette Not Opening?
- Try `Cmd+P` on Mac or `Ctrl+P` on Windows/Linux
- Click the search icon in the header
- Refresh the page if needed

### Where Are My Files?
- Click the arrow on the **left edge** of the screen
- Sidebar slides out with all your files
- Click X to hide it again

### Editor Not Saving?
- Auto-save happens after 2 seconds
- Manual save with "Save" button
- Watch for "Saved X ago" message

### Want Old Layout Back?
- Just keep the sidebar open!
- Pin it with CSS if needed
- All features still accessible
